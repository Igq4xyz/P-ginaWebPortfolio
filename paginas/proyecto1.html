Nube Privada

Implementación de una nube privada doméstica con Nextcloud y Docker

Recientemente desarrollé un proyecto personal con el objetivo de crear una nube privada accesible desde los dispositivos de mi familia, permitiendo almacenar fotografías, documentos y otros archivos dentro de nuestra red local.

1. Instalación inicial en Ubuntu Server

Comencé desplegando Nextcloud en una máquina con Ubuntu Server, configurando manualmente el entorno y verificando que el servicio funcionaba correctamente. Sin embargo, el sistema operativo junto con la instalación completa de Nextcloud utilizaba más de 7 GB de almacenamiento, lo cual me parecía un consumo innecesariamente elevado para este tipo de proyecto.

2. Migración a contenedores con Docker

Para optimizar recursos y facilitar futuras actualizaciones, decidí realizar la instalación mediante Docker. Tras investigar la documentación oficial y varios repositorios, ejecuté la instalación con:

  • Contenedor principal de Nextcloud
  • Backend de base de datos
  • Volúmenes persistentes para datos y configuración

Una vez desplegado, accedí localmente a http://localhost:8080 y comprobé que el servicio respondía correctamente.

3. Configuración de acceso desde dispositivos móviles

Durante las pruebas, detecté que algunos móviles marcaban el dominio interno como "sospechoso", impidiendo el acceso. Para resolverlo entré al contenedor con:

docker exec -it <container_name> bash

Modifiqué el archivo de configuración /config/config.php y añadí el hostname del equipo y una IP estática dentro de la red local: 192.168.1.240

Con esta configuración, el acceso desde cualquier dispositivo de la red fue inmediato.

4. Pruebas de acceso y validación

Pedí a mi padre que accediera desde su móvil mediante http://192.168.1.240:8080 usando las credenciales que había creado previamente. La conexión funcionó sin incidencias y su reacción fue tan positiva que me propuso ampliar el sistema con un disco duro de 1 TB para aumentar la capacidad de almacenamiento. Esta será la siguiente fase del proyecto.
