Proyecto 3: Lavandería Osel

Desarrollo y despliegue de sitio web corporativo

Descripción del Proyecto

A los 16 años completé el diseño, desarrollo y despliegue de una página web corporativa funcional para un negocio local. Durante el verano, una empresa de mi ciudad contactó conmigo para crear una plataforma profesional que presentara su actividad, servicios y datos de contacto de manera clara y accesible.

El proyecto, publicado en lavanderiaosel.es, se llevó a cabo en aproximadamente 2–3 semanas.

Fases del Proyecto

1. Análisis y Toma de Requisitos

  • Revisión del local y flujo de trabajo del negocio
  • Recolección de fotografías y contenido visual
  • Identificación de necesidades: presentación corporativa, formulario de contacto, reseñas y acceso a WhatsApp

2. Diseño y Maquetación

  • Diseño limpio con paleta azul y blanco
  • Técnicas responsive para móvil, tablet y desktop

3. Desarrollo Técnico

  • HTML, CSS y JavaScript
  • Navegación intuitiva y componentes reutilizables
  • Optimización SEO on-page y accesibilidad

4. Despliegue y Alojamiento

  • Configuración de dominio y servidor
  • Pruebas de rendimiento y compatibilidad

Este proyecto marca mi transición hacia proyectos profesionales reales, demostrando capacidad para asumir proyectos completos de principio a fin.
